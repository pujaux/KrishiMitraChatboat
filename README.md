 ## KrishiMitra
## About the Project 
KrishiMitra is an AI-powered bilingual (Hindi + English) chatbot designed to assist farmers with crop recommendations, 
disease diagnosis, soil guidance, and general agricultural queries — with a focus on Madhya Pradesh farming conditions.
## Features
💬 Chat Assistant
Conversational AI for all farming-related queries
Maintains context-aware conversation memory
Provides accurate and simple explanations
## 🌾 Crop Recommendations

## 🎓 Academic Context
This project was developed as part of a college project on AI-powered agricultural assistance. The goal was to create a practical, farmer-friendly tool that:

Requires no technical knowledge to use
Works in the farmer's native language
Provides both offline (rule-based) and online (AI) capabilities
Is deployable for free using GitHub Pages
